# Rhodes

## Rhodesとは

iOSとAndroidのクロスプラットフォーム開発が行える、rubyのSDK

## インストール

※rubyがinstall済み(作業時1.9.0使用)

```console
# gem install Rhodes
```

#### android SDKやNDK等のパスを聞いてくるのでパスを入力してやる
